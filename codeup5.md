__#6046__

십진수를 기반 비트시프트를 통한 2배, 4배, 8배 ... (or 1/2배, 1/4배, 1/8배 ....)

```python
n = int(input())
print(n<<1) #2배
print(n<<2) #4배
print(n<<3) #8배
print(n>>1) #1/2배
print(n>>2) #1/4배
print(n>>3) #1/8배
```
